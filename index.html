Raspjamming OS

Raspjamming OS is focused on using the hardware for GPIO and electronic programming projects. It can be used for C, C# and Python development out of the box. Even for beginners there is the web based visual programming language called Blockly-gPIo. All needed components (web-server , remote service and programming languages with librarys) are preinstalled.
Raspjamming OS is optimized for using at a Raspberry Pi Jam. The boards can be connected directly to a host PC or laptop via USB (OTG-Mode with Zero model) or LAN. No WiFi, Router, Hub or even Internet is needed.
